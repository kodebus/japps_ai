Privacy Policy

Effective date: June 2025 · OmniScript Debug Panel · v1.0

Overview

The OmniScript Debug Panel does not collect, transmit, or share any data. It reads live OmniScript session data directly in your browser and displays it locally. Nothing leaves your environment.

This Privacy Policy describes how the OmniScript Debug Panel (the "Component"), developed by Japps.ai, handles information when installed and used within a Salesforce org. Our approach is simple: we do not collect your data.

Information We Do Not Collect

The OmniScript Debug Panel does not collect, store remotely, or transmit any of the following:

  • Salesforce org data, record IDs, or field values
  • OmniScript JSON data or session state
  • User credentials, profiles, or account information
  • Usage analytics, telemetry, or behavioral data
  • Device information or IP addresses

How the Component Works

The OmniScript Debug Panel is a Lightning Web Component (LWC) that reads the omniJsonData object from the active OmniScript step at runtime. This data is displayed directly within your Salesforce session and is never transmitted outside of it.

  • All data displayed in the panel exists only within your active browser session
  • No data is written to any external server, database, or API
  • The Copy JSON button copies data to your local clipboard only
  • Removing the component from a step leaves no data behind

Third-Party Services

The OmniScript Debug Panel does not integrate with, send data to, or rely on any third-party analytics, advertising, tracking, or data collection services. There are no external API calls made by this component.

Data Security

Since the component operates entirely within your Salesforce org and browser session, data security is governed by your org's existing Salesforce security model — including profiles, permission sets, and field-level security. Japps.ai does not have access to your org or its data.

Open Source

The OmniScript Debug Panel is open source and available for review on GitHub at github.com/kodebus/omniDebugPanel. You are welcome to inspect the full source code to verify these privacy claims directly.

Changes to This Policy

If this Privacy Policy is updated, the new version will be posted at this URL with a revised effective date. We do not anticipate changes that would introduce any form of data collection.
