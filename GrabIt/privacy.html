Privacy Policy

Effective date: June 2025  ·  Grab It Chrome Extension

Overview

Grab It does not collect, transmit, or share any personal data. All information you enter is stored locally in your browser and never leaves your device.

This Privacy Policy describes how Grab It (the "Extension"), developed by Japps.ai, handles information when you use the extension. We are committed to full transparency — and in this case, the simplest summary is the complete one: we do not collect your data.

Information We Do Not Collect

Grab It does not collect, store remotely, or transmit any of the following:

  • Personal identifying information (name, email, account credentials)
  • Salesforce org data, component names, or metadata
  • Work log entries, task descriptions, or story numbers
  • Browsing history or page content beyond what you explicitly log
  • Device information, IP addresses, or location data
  • Usage analytics or behavioral data

Local Storage

Grab It uses Chrome's built-in chrome.storage.local API to save your work log entries, settings, and report data directly on your device. This data:

  • Is stored only in your browser — never on any external server
  • Is isolated per Chrome profile and not shared between users
  • Can be exported or deleted by you at any time from within the extension
  • Is automatically removed if you uninstall the extension

Permissions and Why We Need Them

Grab It requests the following Chrome permissions, each used only for its stated purpose:

  • storage — to save your work log entries and settings locally in your browser
  • activeTab — to read the current Salesforce page when you click the extension icon, in order to detect the component you are working on
  • scripting — to extract component metadata (name, version, description) from the active Salesforce designer page, only when triggered by you
  • host permissions (*.salesforce.com, *.lightning.force.com, *.force.com) — to enable component detection within Salesforce org pages only

No permission is used to collect, monitor, or transmit data in the background or without your direct action.

Third-Party Services

Grab It does not integrate with, send data to, or rely on any third-party analytics, advertising, tracking, or data services. There are no third-party SDKs, remote scripts, or external API calls made by this extension.

Remote Code

Grab It does not load or execute any remote code. All JavaScript is bundled and included locally within the extension package at the time of installation. No code is fetched or evaluated from external sources at runtime.

Children's Privacy

Grab It is a professional productivity tool intended for use by Salesforce developers. It is not directed at children under the age of 13, and we do not knowingly collect any information from minors.

Changes to This Policy

If this Privacy Policy is updated, the new version will be posted at this URL with a revised effective date. Continued use of the extension after any changes constitutes acceptance of the updated policy. We do not anticipate changes that would introduce data collection of any kind.
